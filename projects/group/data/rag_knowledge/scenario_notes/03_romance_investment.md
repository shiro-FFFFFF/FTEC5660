# Scenario Note: 03_romance_investment

## Scenario Type
Romance / investment trust-building scam

## Event Flow
- extended chat builds trust
- investment narrative introduced
- victim is encouraged to transfer funds
- risk emerges from long-term context, not a single alarming message

## Key Risk Idea
Historical context matters more than any single event.

## Relevant Scam Patterns
- romance scam
- investment fraud
- emotional manipulation
